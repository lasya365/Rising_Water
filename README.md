# Rising Waters: Flood Prediction System

Rising Waters is an AI-powered web application designed to predict flood risks based on meteorological parameters. This project utilizes machine learning models to analyze rainfall and cloud cover data, providing actionable insights into potential flood events.

## 🚀 Project Overview
This project was developed as a comprehensive AI/ML pipeline, covering:
- **Data Preprocessing:** Cleaning and scaling raw meteorological data.
- **Model Building:** Training and evaluating various algorithms (XGBoost, Random Forest, etc.) to achieve high predictive accuracy.
- **Web Deployment:** A responsive interface built with Flask that allows users to input real-time weather parameters and receive instant risk assessments.

## 🛠 Technologies Used
- **Language:** Python
- **Framework:** Flask (Web UI)
- **Machine Learning:** Scikit-Learn, XGBoost
- **Data Manipulation:** Pandas, NumPy
- **Storage:** Joblib (Model & Scaler serialization)

## 📋 Features
- **Intuitive UI:** User-friendly form for inputting weather variables.
- **Predictive Engine:** Real-time classification of flood risk ("Safe" vs. "High Risk").
- **Scalable Architecture:** Pre-processing and model inference pipelines integrated into a backend Flask application.
